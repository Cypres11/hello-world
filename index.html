Import contacts

Mac Contacten → vCard (.vcf)

  1. Open the Contacts app on your Mac
  2. Select all contacts with ⌘A
  3. Go to File → Export → Export vCard…
  4. Save the .vcf file and upload it below

CSV file (.csv)
